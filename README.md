# 运动场地预约平台

纯前端运动场地预约系统，数据存储在浏览器 localStorage 中，无需后端服务。

## 技术栈

- **框架**：React 18 + TypeScript
- **构建工具**：Vite 6
- **数据持久化**：localStorage
- **样式**：内联 CSS（写在 index.html 的 style 标签中）
- **无第三方 UI 库**

## 启动方式

```bash
# 安装依赖
npm install

# 启动开发服务器（端口 5173）
npm run dev
```

打开浏览器访问 `http://localhost:5173`

## 功能说明

### 场地浏览
- 首页以卡片形式展示所有场地，含图片、名称、类型标签和位置
- 支持按类型筛选（篮球/羽毛球/乒乓球/网球/足球）
- 支持按名称、位置、类型关键字搜索

### 场地详情
- 场地照片轮播（自动播放 + 手动切换）
- 详细的场地信息：地址、价格、容量、评分、营业时间、联系电话
- 设施服务标签展示
- **日历视图**：以周为单位查看未来 14 天的预约情况
- **时段选择**：每天分为上午/下午/晚上三个时段，可选（绿色）、已满（灰色）

### 预约流程
1. 在场地详情页选择日期和时段
2. 填写预约人数、联系人姓名和手机号
3. 确认提交后自动生成 6 位签到码

### 签到核销
- 预约成功后展示签到码（数字码 + 二维码图形）
- 到场出示签到码进行核销
- 支持模拟签到操作

### 我的预约
- 三个 Tab：待签到、已完成、已取消 + 全部
- 支持取消未使用的预约（自动恢复时段可预约状态）
- 已完成的预约支持跳转评价

### 场地评价
- 五星评分 + 文字评价
- 评价提交后自动更新场地平均评分
- 场地详情页底部展示所有评价

### 场地管理（管理端）
- 在个人中心开启"场地方身份"后，导航栏出现"场地管理"入口
- 查看今日待签到预约列表，可快速确认签到
- 编辑场地信息（名称、地址、价格、容量、设施等）

### 个人中心
- 编辑个人信息（昵称、手机号）
- 切换场地方身份

## 预置数据

首次打开自动生成以下内容：
- **8 个运动场地**：2个篮球馆、2个羽毛球馆、2个乒乓球馆、1个网球场、1个足球场
- **14 天时段数据**：随机生成部分时段已约满的状态
- **4 条预约记录**：3条已完成 + 1条待签到
- **5 条场地评价**

## 项目结构

├── package.json
├── index.html          # 入口 HTML + 全局样式
├── vite.config.ts
├── tsconfig.json
├── README.md
└── src/
